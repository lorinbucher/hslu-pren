## Installation

```shell
# create virtual environment
python3 -m venv ~/.virtualenvs/hslu-pren
# initialize virtual environment
source ~/.virtualenvs/hslu-pren/bin/activate
# install dependencies
python3 -m pip install -r requirements.txt
```

## Deployment

```shell
source ~/.virtualenvs/hslu-pren/bin/activate
python3 main.py
```
